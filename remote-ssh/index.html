【SSH】使用ssh连接个人win10电脑

发表于2021-12-18|更新于2021-12-18|help
|阅读量:

前言

本文旨在将个人的一台不便于随身携带的win10主机打造成一个服务器，使得可以在其他地方使用SSH远程连接该主机进行操作。

1 配置SSH服务

1.1 安装OpenSSH Server

1.打开“设置”，点击“应用”：

2.点击“应用和功能”里的“可选功能”：

3.在“可选功能”中点击“添加功能”：

4.搜索“OpenSSH服务器”并安装。

1.2 开启SSH服务

1.右键我的电脑，选择“管理”：

2.在“服务和应用”中选择“服务”，找到“OpenSSH Authentication Agent”和“OpenSSH SSH Server”：

3.分别右键它们，点击“属性”，将“启动类型”修改为“自动”， 然后点击“启动”来启动这个服务：

1.3 验证SSH是否配置成功

打开cmd，输入

如果提示你输入密码，就说明之前的操作都成功了。

2 免密ssh登录

开启SSH服务后，意味着网络上任何人都可以通过SSH来远程访问你的电脑了，所以建议将自己的用户密码改为强密码。

但是每次SSH登陆都需要输入强密码实在是有点“懒人震怒”，我可以通过配置SSH密钥来实现免密登陆。

2.1 生成SSH密钥对

以mac电脑为例，在终端输入：

以RSA为加密算法生成SSH密钥对，默认密钥存放位置为：私钥 ~/.ssh/id_rsa ，公钥 ~/.ssh/id_rsa.pub 。

通过切换管理员权限将公钥复制出来到桌面：

2.2 将公钥上传到SSH服务器指定位置

根据用户类别，分为标准用户和管理员用户，它们需要公钥放置的位置并不相同。

2.2.1 标准用户

将公钥放在 C:\Users\username\.ssh\ 目录下，并将公钥重命名为 authorized_keys 。

2.2.2 管理员用户

将公钥放在 C:\ProgramData\ssh\ 目录下，并将公钥重命名为 administrators_authorized_keys 。

1.右键我的电脑，选择“管理”：

2.在“服务和应用”中选择“服务”，找到“OpenSSH Authentication Agent”和“OpenSSH SSH Server”：

3.分别右键它们，点击“重启”，这样就OK啦。
